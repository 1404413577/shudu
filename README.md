# 🧩 数独游戏 (Sudoku Game)

基于 Vue 3 + Vite 开发的现代化数独游戏，支持多种难度、智能笔记、撤销功能等高级特性。

## ✨ 功能特色

### 🎮 核心游戏功能
- **三种难度等级**：简单(40个空格)、中等(50个空格)、困难(60个空格)
- **智能数独生成**：使用回溯算法生成唯一解数独谜题
- **实时验证**：即时检测数字冲突并高亮显示
- **自动求解**：一键获取完整解答

### 🛠️ 高级功能
- **↶ 撤销功能**：支持多步撤销操作
- **🧽 擦除模式**：快速清除单元格数字
- **📝 笔记功能**：在空格中添加候选数字提示
- **💡 智能提示**：随机填入正确答案作为提示

### 🎨 用户体验
- **响应式设计**：完美适配桌面端和移动端
- **实时计时**：记录游戏用时
- **状态管理**：游戏中、已完成、有错误等状态显示
- **视觉反馈**：数字高亮、错误标记、选中效果

## 🎯 游戏操作

### 基本操作
- **选择单元格**：点击空白单元格
- **输入数字**：直接输入1-9的数字
- **删除数字**：按Delete或Backspace键
- **键盘导航**：使用方向键在单元格间移动

### 功能按钮
- **新游戏**：开始新的数独游戏
- **撤销**：撤销上一步操作
- **擦除**：进入擦除模式，点击格子清空数字
- **笔记**：进入笔记模式，添加候选数字
- **提示**：自动填入一个正确答案
- **求解**：显示完整解答
- **检查**：验证当前答案
- **清空**：清除所有用户输入

### 核心算法说明

#### 数独生成算法
1. 创建空的9x9网格
2. 使用回溯算法填充完整数独
3. 随机移除指定数量的数字
4. 验证唯一解

#### 数独验证算法
1. 检查行冲突：同行不能有重复数字
2. 检查列冲突：同列不能有重复数字
3. 检查宫格冲突：同3x3宫格不能有重复数字
